# 📊 Synthetic User Dataset Generator

This project generates a synthetic dataset of users with realistic attributes and relational structure. The goal is to simulate a small database table that includes **primary keys, foreign keys, and meaningful categorical data**, making it useful for testing, learning, and prototyping data workflows.

---

## 🔍 Overview

The script creates 100 user records with attributes such as name, age, nationality, favorite sport, and marital status. It mimics relational database behavior by linking users to predefined tables (countries and sports) through identifiers, simulating **foreign key relationships**.

Each record is built using random but coherent values, ensuring that the dataset maintains logical consistency while still being diverse.

---

## ⚙️ Features

- Automatic dataset generation (100 rows)  
- Simulated **primary and foreign keys**  
- Realistic random data (names, ages, categories)  
- Structured output using a DataFrame  
- Export to CSV format  

---

## 🎯 Use Cases

- Data analysis practice  
- Database design demonstrations  
- Machine learning preprocessing  
- Testing ETL pipelines  

---

## 🚀 Output

The final dataset is saved as a CSV file and includes a preview of the first rows, providing a quick look at the generated data structure.
